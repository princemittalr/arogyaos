# Changelog
All notable changes to ArogyaOS will be documented in this file.

## [0.1.0] - 2026-07-06
### Added
- Vitest unit testing framework with test configuration.
- 31 unit tests validating models, schemas, and helper logic.
- CI/CD workflow validating linting, type-checking, tests, and production builds.
- Structured accessibility enhancements for screen readers, keyboard paths, and interactive sorting tables (WCAG 2.2 AA).
- Textual summaries for all data visualization charts.

### Changed
- Recharts and dynamic components decoupled from primary bundles.
- Strengthened Firebase rules and PII data sanitization workflows.
- Improved logging and error reporting systems.
