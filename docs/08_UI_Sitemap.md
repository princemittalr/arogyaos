# Navigation Rules

- Maximum 3 clicks to reach any feature.
- Persistent sidebar on desktop.
- Collapsible sidebar on tablet.
- Bottom navigation for citizen mobile experience.
- Consistent header across all dashboards.

---

# Design Principles

- Clean healthcare-focused UI
- Minimal cognitive load
- Accessibility-first
- High contrast
- Role-specific navigation
- Consistent spacing
- Fast loading
- Real-time updates
