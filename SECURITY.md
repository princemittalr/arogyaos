# Security Policy

## Supported Versions
Only the latest release of ArogyaOS is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability
We take the security of ArogyaOS very seriously. If you spot a vulnerability, please do NOT create a public issue. Instead, report it privately:

1. Email the core team at `security@arogyaos.in`
2. Provide details of the vulnerability, including steps to reproduce, impact, and potential remediation.
3. We will acknowledge receipt of your report within 48 hours and work with you to patch and coordinate a disclosure timeline.
